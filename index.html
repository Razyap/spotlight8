Введение

Цели, задачи и структура пособия

Настоящее методическое пособие разработано в качестве дополнительного ресурса для учителей английского языка, работающих по УМК «Английский в фокусе» (Spotlight 8) в общеобразовательных организациях Российской Федерации. Пособие реализует концепцию трёхэтапного обучения аудированию с опорой на современные интерактивные цифровые платформы, доступные на территории РФ.

🎯 Цели пособия

  • Создать банк аудирования по каждому модулю Spotlight 8
  • Систематизировать платформы по этапам работы с аудиотекстом
  • Обеспечить учителя готовой структурой занятия
  • Повысить мотивацию через геймификацию

📋 Каждый раздел включает

  • Тематику и жанр аудиотекста
  • Таблицу заданий по трём этапам
  • Платформу и тип задания
  • Ссылку на ресурс
  • Методическое примечание

✅ Критерии отбора платформ

  • Доступность на территории РФ
  • Бесплатный функционал
  • Поддержка форматов заданий
  • Простота для учителя и ученика
  • Обратная связь и геймификация
  • Возраст 13–14 лет, уровень B1

Платформы и ресурсы

Все платформы доступны на территории РФ без средств обхода блокировок

Quizlet Предтекстовый
quizlet.com ↗

Международная платформа для изучения лексики с интерактивными карточками, играми и тестами. Режимы: Flashcards, Learn, Test, Match, Gravity, Quizlet Live (командный). Готовые наборы по темам Spotlight 8 находятся через поиск.

Wordwall Предтекстовый
wordwall.net ↗

Платформа для интерактивных обучающих игр. Ключевая особенность — переключение между шаблонами одним кликом: Quiz, Match up, Anagram, Crossword, Whack-a-mole, Maze chase и более 30 других шаблонов.

ISLcollective Текстовой
en.islcollective.com ↗

Международный ресурс с базой рабочих листов и интерактивных упражнений по английскому языку для уровней A2–B1. Регистрация бесплатна. Материалы можно адаптировать под конкретный урок.

РЭШ Текстовой
resh.edu.ru ↗

Российская электронная школа — федеральный портал с полным курсом интерактивных уроков (1–11 класс) по ФГОС. По английскому языку: видеоуроки, тренировочные и проверочные задания. Доступ бесплатный.

MyQuiz Послетекстовый
myquiz.ru ↗

Российская платформа для интерактивных викторин в режиме реального времени. Ученики подключаются по коду без регистрации. Поддерживает командную и индивидуальную игру, режим домашнего задания. Интерфейс на русском языке.

Online Test Pad Послетекстовый
onlinetestpad.com ↗

Российский сервис для тестов, опросов и кроссвордов. Форматы: одиночный/множественный выбор, соответствие, ввод ответа, расстановка по порядку. Результаты автоматически проверяются и доступны в кабинете учителя.

Методические рекомендации

Для учителя — как работать с пособием и платформами эффективно

①
Общие принципы

Каждый раздел — для одного урока или серии по модулю. Соблюдайте трёхэтапную структуру. Временны́е рамки: предтекстовый — 5–7 мин., текстовой — 10–15 мин., послетекстовый — 7–10 мин.

②
Quizlet / Wordwall

Создайте или найдите набор карточек по теме модуля заранее. Используйте Quizlet Live в начале урока как «разогрев» — это сразу создаёт соревновательную атмосферу.

③
ISLcollective

Перед уроком проверьте доступность выбранного видео. При необходимости подготовьте собственный рабочий лист. Для классов с низким уровнем допускается двукратное прослушивание.

④
MyQuiz / Online Test Pad

Подготовьте викторину заранее и протестируйте код подключения. Для командной игры разделите класс на группы по 3–4 человека. Победившая команда получает бонусные баллы.

⑤
Дифференциация

Слабые учащиеся: субтитры, расширенный режим Quizlet. Сильные: добавьте письменное резюме или монолог по видео. Разноуровневый класс: базовые карточки vs режим Gravity (Quizlet).

⑥
Инклюзия

На платформах, поддерживающих субтитры — включайте их для учащихся с трудностями восприятия на слух. Quizlet позволяет изучать карточки в индивидуальном темпе.

Критерии оценки

Рекомендуемая система оценивания выполнения заданий (максимум 10 баллов)

Критерий Баллы Описание
Понимание основного содержания 0–3 Учащийся правильно определяет тему, главную идею и структуру текста
Понимание деталей 0–3 Учащийся извлекает конкретную информацию: факты, имена, числа, даты
Выполнение заданий на платформе 0–2 Правильность выполнения интерактивного задания (gap-fill, тест, кроссворд)
Активность в послетекстовой работе 0–2 Участие в командной викторине или обсуждении, качество ответов
